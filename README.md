# mugatu_motion

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mugatu_motion'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mugatu_motion

Then require the gem in your RubyMotion Gemfile:

    require 'mugatu_motion'
